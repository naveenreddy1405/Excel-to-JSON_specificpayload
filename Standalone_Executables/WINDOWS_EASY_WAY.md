# Windows - Easiest Way to Use (No Typing!)

**For Windows users - Just drag and drop, no commands needed!**

---

## **Step 1: First Run**

Double-click `Convert.bat`

A Command Prompt window opens with instructions.

---

## **Step 2: Use It (Every Time)**

### **Just follow the prompts:**

1. **Double-click** `Convert.bat`
2. Command Prompt opens with instructions
3. **Drag your Excel file** onto the Command Prompt window
4. Press **Enter**
5. ✅ Done! JSON created!

---

## **Example:**

```
Drag your Excel file here (then press ENTER): [DRAG FILE HERE]
```

Just drag your file onto the Command Prompt and press Enter!

---

## **Output**

Your JSON file appears in the **same folder** as your Excel file.
