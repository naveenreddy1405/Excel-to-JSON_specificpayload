# Mac - Easiest Way to Use (No Typing!)

**For Mac users - Just drag and drop, no commands needed!**

---

## **Step 1: One-Time Setup**

Run this in Terminal (just once):

```bash
chmod +x ~/Downloads/Excel-to-JSON_specificpayload-main/Standalone_Executables/Convert.sh
```

Or drag the file and use the quarantine fix:
```bash
xattr -d com.apple.quarantine ~/Downloads/Excel-to-JSON_specificpayload-main/Standalone_Executables/Convert.sh
```

---

## **Step 2: Use It (Every Time)**

### **That's it! Just follow the prompts:**

1. **Double-click** `Convert.sh`
2. Terminal opens with instructions
3. **Drag your Excel file** onto the Terminal window
4. Press **Enter**
5. ✅ Done! JSON created!
