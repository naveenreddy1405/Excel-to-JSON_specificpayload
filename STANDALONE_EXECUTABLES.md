## **🚀 MAC USERS - HOW TO USE**

### **Method 1: Double-Click (Easiest)**

1. Find `ExcelToJsonConverter` file
2. Double-click it
3. It opens Terminal
4. Drag your Excel file into the Terminal window
5. Press Enter
6. ✅ JSON created!

### **Method 2: Terminal Command**

```bash
./ExcelToJsonConverter input.xlsx
./ExcelToJsonConverter input.xlsx output.json
./ExcelToJsonConverter input.xlsx output.json -v --pretty
```

## **📥 How to Build Windows .EXE on Your PC**

If you need to create the Windows .exe:

1. Download the source files from GitHub
2. Install Python
3. Run: `python -m pip install pyinstaller`
4. Run: `pyinstaller --onefile --console excel_to_json_converter.py`
5. Find `ExcelToJsonConverter.exe` in the `dist` folder
