# Excel to JSON Converter - Shared Drive Tool

**Everything you need is in this folder. Choose your operating system below.**

---

## **📦 Quick Start**

### **Windows Users:**
1. Open the **`Windows/`** folder
2. **Double-click `run.bat`**
3. Drag your Excel file onto the window
4. ✅ Get your JSON file instantly!

### **Mac/Linux Users:**
1. Open the **`Mac_Linux/`** folder
2. Open Terminal in that folder
3. Run: `chmod +x run.sh` (one-time setup)
4. Run: `./run.sh your_file.xlsx`
5. ✅ Get your JSON file!
